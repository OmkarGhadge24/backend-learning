# Backend Learning Journey

This repository documents my progressive learning in backend development using Node.js, Express.js, and MongoDB. Each folder represents a milestone in building increasingly complex applications, starting from basic concepts and advancing to full-stack projects with authentication, file handling, and testing.

## Project Structure

### 1. Basics
**Focus:** Fundamental Express.js setup and basic API operations  
**Technologies:** Express.js  
**Features:**
- Simple in-memory notes API (CRUD operations)
- Basic server configuration
- JSON request/response handling

### 2. Authentication
**Focus:** User authentication and authorization  
**Technologies:** Express.js, MongoDB, JWT, bcrypt  
**Features:**
- User registration and login
- JWT token-based authentication
- Cookie handling
- Password hashing
- Protected routes

### 3. Notes App
**Focus:** Database integration and data persistence  
**Technologies:** Express.js, MongoDB, Mongoose  
**Features:**
- MongoDB connection and schema design
- CRUD operations with database persistence
- Mongoose ODM usage

### 4. Media Project
**Focus:** File uploads and full-stack development  
**Technologies:** Express.js, MongoDB, Multer, ImageKit, React, Vite  
**Features:**
- Image upload and storage
- Full-stack application (Backend + Frontend)
- CORS handling
- React frontend with Tailwind CSS
- Social media-style post creation

**Frontend:** React app with routing and API integration

### 5. Spotify
**Focus:** Complex application with multiple entities  
**Technologies:** Express.js, MongoDB, JWT, Multer, ImageKit, bcrypt  
**Features:**
- User authentication
- Music and album management
- File upload for music/images
- Complex data relationships
- Advanced API design

### 6. Testing
**Focus:** API testing and validation  
**Technologies:** Express.js, Jest, Supertest, express-validator  
**Features:**
- Unit and integration testing
- Input validation middleware
- Test-driven development practices
- API endpoint testing

## Technologies Used

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB with Mongoose ODM
- **Authentication:** JWT, bcrypt
- **File Handling:** Multer, ImageKit
- **Testing:** Jest, Supertest
- **Frontend:** React, Vite, Tailwind CSS
- **Validation:** express-validator
